## Semantic tags

```
<body>

<header>

</header>

<aside>

</aside>

<main>

<section>

</section>

</main>

<footer>

</footer>

</body>
```

### Purpose

| Tag | Purpose |
|------|---------|
| `<header>` | Top navigation or introductory content |
| `<nav>` | Navigation links |
| `<aside>` | Sidebar or supplementary content |
| `<main>` | Main content of the page |
| `<section>` | A logical section of content (e.g., Productivity, Features) |
| `<article>` | Standalone, independent content |
| `<footer>` | Footer of the page |


### Example

```
<body>

<header>

</header>

<aside>

</aside>

<main>

<section class="overview">

</section>

<section class="productivity">

</section>

<section class="developer-tools">

</section>

<section class="ai-tools">

</section>

</main>

</body>
```
